# Dataset Pengumpulan dan Review Putusan Pengadilan Narkotika dan Psikotropika

Selamat datang di repositori **Dataset Pengumpulan dan Review Putusan Pengadilan Narkotika dan Psikotropika**! Repositori ini berisi pengumpulan dataset dalam bentuk file ZIP yang berisi dokumen PDF putusan pengadilan terkait kasus narkotika dan psikotropika, serta file overview dalam format XLSX. Dataset ini dikumpulkan sebagai bagian dari tugas pengumpulan dan review dataset untuk keperluan pendidikan atau riset untuk konteks hukum.

## Tentang Dataset

Dataset ini terdiri dari koleksi dokumen PDF putusan pengadilan negeri terkait tindak pidana narkotika golongan I (seperti sabu/shabu) berdasarkan Undang-Undang Nomor 35 Tahun 2009 tentang Narkotika. Tujuan pengumpulan adalah untuk melakukan review terhadap barang bukti, amar putusan, dan elemen hukum lainnya. Dataset ini berguna untuk analisis hukum, riset kebijakan, atau studi kasus di bidang kriminologi dan hukum pidana.

Beberapa fitur utama dari repositori ini:
- **Koleksi PDF dalam ZIP**: Dokumen putusan pengadilan yang difokuskan pada kasus narkotika dan psikotropika.
- **Overview List**: Ringkasan dalam bentuk file XLSX yang mencakup detail seperti nomor putusan, lembaga peradilan, barang bukti, dan amar putusan.
- **Dokumentasi Lengkap**: Panduan akses, penggunaan, dan kontribusi untuk memudahkan kolaborasi dan review lebih lanjut.

Topik utama: Kasus pidana narkotika di Pengadilan Negeri Padang (dan sejenisnya), dengan fokus pada kepemilikan, penjualan, atau peredaran narkotika golongan I bukan tanaman.

## Persyaratan

Untuk mengakses dan menggunakan dataset:
- GitHub account untuk clone repositori.
- PDF viewer (misalnya, Adobe Acrobat atau browser seperti Chrome).
- Spreadsheet viewer (misalnya, Microsoft Excel atau Google Sheets).
- Opsional: Tools seperti Python dengan library `PyPDF2` untuk ekstraksi teks dari PDF, atau `pandas` untuk analisis XLSX.

## Instalasi

1. Clone repositori ini:
   ```
   git clone https://github.com/Zero4427/dataset-narkotika-psikotropika.git
   ```

2. Masuk ke direktori:
   ```
   cd dataset-narkotika-psikotropika
   ```

3. Ekstrak ZIP (jika diperlukan):
   - Gunakan tools seperti `unzip Narkotika dan Psikotropika.zip` di terminal, atau ekstrak secara manual.

4. (Opsional) Instal dependensi untuk analisis:
   ```
   pip install PyPDF2 pandas openpyxl
   ```

## Penggunaan

Untuk melihat dataset:
- Buka folder `Dataset/` dan ekstrak ZIP untuk mengakses PDF. Buka PDF secara manual untuk membaca putusan lengkap.
- Buka folder `Overview/` dan buka `Overview.xlsx` untuk melihat ringkasan review.

Untuk analisis overview:
- Import XLSX ke Google Sheets atau Excel untuk filtering atau visualisasi data.

## Review Dataset

Berdasarkan tugas, berikut ringkasan review umum:
- **Kualitas Data**: Dataset lengkap dengan dokumen resmi putusan pengadilan, relevan untuk analisis hukum narkotika. Semua kasus melibatkan sabu/shabu dengan berat bervariasi, dan putusan berdasarkan UU Narkotika.
- **Kelebihan**: Format PDF standar, mudah diakses; overview XLSX memudahkan pencarian cepat.
- **Kekurangan**: Beberapa PDF mungkin scanned (memerlukan OCR jika teks tidak ekstrakable); data terbatas pada kasus spesifik di Padang tahun 2025.
- **Rekomendasi**: Gunakan untuk riset pola barang bukti atau tren hukuman di kasus narkotika. Bisa dikembangkan dengan menambahkan lebih banyak kasus.

Detail review per kasus ada di tabel overview di atas.
